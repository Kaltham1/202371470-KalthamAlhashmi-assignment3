# Personal Portfolio – Assignment 3

## 📌 Project Description
This project is an advanced version of my personal portfolio website.  
It builds on previous assignments by adding dynamic features, API integration, and improved interactivity using JavaScript.

The goal of this assignment was to demonstrate advanced web development concepts, including external data fetching, complex logic, and state management.

---

## 🚀 Features

- GitHub API Integration  
  Dynamically fetches and displays repositories from my GitHub profile.

- Project Filtering and Sorting  
  Allows users to filter projects by category and sort them by date.

- Dark / Light Mode  
  Users can toggle between themes, and their preference is saved using localStorage.

- Contact Form Validation  
  Ensures user input is valid before submission and provides feedback messages.

- Responsive Design  
  Works across different screen sizes with clean UI and animations.
