# Technical Documentation

## 📁 Project Structure
- index.html – Main HTML structure  
- css/styles.css – Styling and layout  
- js/script.js – Interactivity and logic  
- assets/ – Images and files  
- docs/ – Documentation files  

---

## 🔧 Features

### 1. API Integration
- Integrated the GitHub API to dynamically fetch repositories  
- Displayed repository name, description, languages, and last updated date  
- Implemented error handling for failed API requests  

---

### 2. Complex Logic
- Implemented project filtering based on category  
- Implemented sorting based on project date  
- Combined filtering and sorting logic for dynamic updates  

---

### 3. State Management
- Implemented dark/light mode toggle  
- Used localStorage to save user preference  
- Restored the selected theme on page reload  

---

### 4. Data Handling & Validation
- Contact form includes:
  - Required field checks  
  - Email format validation  
  - Phone number validation  
- Displays user-friendly error and success messages  

---

## ⚡ Performance Optimization
- Compressed and optimized images (reduced to under 250 KB)  
- Removed unnecessary code and unused styles  
- Used efficient DOM updates
